## Contracts -- MERN Application
---

## Backend Tech Stack
> Node  
> Express  
> MongoDB  
> Mongoose  

## Frontend Tech Stack
> React  
> Redux  
> Redux Thunk  
> Bootstrap  

# Commands
```
Start the `MongoDB` server `mongodb://localhost:27017/contracts`

cd backend
npm install
npm start

Server Starts Listening on http://localhost:4000

cd frontend
npm install
npm start

Client Starts Listening on http://localhost:3000
